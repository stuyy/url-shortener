# URL Shortener

This is a simple URL Shortener project that I coded live on stream over on my [YouTube Channel](https://www.youtube.com/c/ansonthedeveloper)

# Installation

- Clone this repository
- Run `npm i` or `yarn install`
- Make sure you have MongoDB Installed as well
- Run `npm run dev` or `yarn dev`
- Or if you want to run the compiled version, run `npm run build` or `yarn build` then run `npm start` or `yarn start`
